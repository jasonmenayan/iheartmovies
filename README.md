# iheartmovies

To install and run:

```
git clone https://github.com/jasonmenayan/iheartmovies.git
cd iheartmovies
npm install
node src/server/server.js
```

**Node v6 required**. If using v4, please execute instead:

`node --harmony src/server/server.js`

Then open:

`http://localhost:8080/`

in your browser.
